## 🛠️ Common Tasks

You can perform several actions with this tool. Here are the most common ways to use it.

### Saving from Social Media
If you want to save a video from Instagram or Reddit, copy the link from your web browser. Paste that link after the download command provided by the application. The tool finds the video file, cleans the data, and saves it to your computer folder.

### Resuming Downloads
Sometimes your internet connection drops. Media-puller tracks your downloads. You can stop a session and start again later. The software remembers where it left off and completes the file transfer without starting over.

### Managing Folders
By default, the program groups files by the website name. This keeps your media library organized. You can change the folder location in the settings menu if you prefer to save files to an external drive or a specific folder on your desktop.

## 💡 Tips for Better Results

*   Check your connection before you start large batches of downloads.
*   Keep your system free of heavy background tasks while downloading long videos.
*   Ensure you have enough space on your hard drive before you begin.
*   Read the error messages in the event a download fails. Often, these messages explain that the link is private or the video was deleted from the website.

## 🛡️ Privacy and Safety

This software runs locally on your machine. We do not store your personal browsing data or account passwords. All media files go directly from the host website to your computer. This ensures that you maintain control over your content. The program acts as a bridge between the source website and your disk drive.

## ❓ Troubleshooting

If you encounter issues, look at these common fixes:

*   The window closes immediately: Verify that you installed the correct version for Windows.
*   Download fails to start: Check if the link you pasted is correct or if the website requires you to be logged in to view the content.
*   Files are missing: Check the default folder location mentioned in the settings menu.
*   Slow speeds: This depends entirely on your internet service provider and the speed of the website hosting the media.

If these steps do not fix your issue, check the official project page for recent updates or known bugs. Developers update the software frequently to handle changes on websites like Instagram or Reddit. Always keep your software current to ensure the best performance.
